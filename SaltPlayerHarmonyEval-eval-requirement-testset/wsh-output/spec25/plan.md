# 悬浮窗状态栏歌词SPEC

设置-歌词-悬浮窗状态栏歌词功能，用户可通过开关启用悬浮窗在状态栏区域滚动展示当前播放歌词，并可自定义歌词的位置、宽度、字体大小、颜色、对齐方式等参数。

## 场景一

### 场景概述

用户打开悬浮窗状态栏歌词开关，但系统未授予悬浮窗权限

### 场景逻辑步骤

- 1. 用户进入"设置-歌词-悬浮窗状态栏歌词"页面
- 2. 用户点击悬浮窗状态栏歌词主开关（当前为关闭状态）
- 3. 系统检测到未授予悬浮窗权限
- 4. 弹出提示告知用户需要授予悬浮窗权限，并引导用户前往系统设置页面授权
- 5. 开关保持关闭状态，悬浮窗不显示

## 场景二

### 场景概述

用户打开悬浮窗状态栏歌词开关，系统已授予悬浮窗权限，悬浮窗正常显示

### 场景逻辑步骤

- 1. 用户进入"设置-歌词-悬浮窗状态栏歌词"页面
- 2. 用户点击悬浮窗状态栏歌词主开关（当前为关闭状态）
- 3. 系统检测到已授予悬浮窗权限
- 4. 开关切换为打开状态
- 5. 在屏幕状态栏区域创建悬浮窗，使用默认参数展示：左右位置0%、上下位置0px、宽度150dp、字体大小14.0dp、颜色蓝色、左对齐
- 6. 如果当前正在播放歌曲，悬浮窗立即开始滚动展示当前歌词行
- 7. 如果当前未在播放歌曲，悬浮窗创建但歌词文本不可见，等待播放开始

## 场景三

### 场景概述

用户关闭悬浮窗状态栏歌词开关

### 场景逻辑步骤

- 1. 用户进入"设置-歌词-悬浮窗状态栏歌词"页面
- 2. 用户点击悬浮窗状态栏歌词主开关（当前为打开状态）
- 3. 开关切换为关闭状态
- 4. 悬浮窗从屏幕上移除，不再显示歌词

## 场景四

### 场景概述

播放歌曲时，悬浮窗实时滚动展示当前歌词

### 场景逻辑步骤

- 1. 悬浮窗状态栏歌词开关已打开，悬浮窗已显示
- 2. 用户正在播放一首有歌词的歌曲
- 3. 随着播放进度推进，当前歌词行发生变化时，悬浮窗中的歌词文本实时更新为当前歌词行
- 4. 当歌词文本长度超出悬浮窗设定宽度时，歌词以跑马灯方式从右向左水平滚动展示完整内容
- 5. 当歌词文本长度未超出悬浮窗设定宽度时，歌词静态展示，不触发滚动

## 场景五

### 场景概述

暂停播放时，悬浮窗歌词淡出隐藏

### 场景逻辑步骤

- 1. 悬浮窗状态栏歌词开关已打开，当前正在播放歌曲，悬浮窗正在展示歌词
- 2. 用户暂停播放
- 3. 悬浮窗中的歌词文本以淡出动画隐藏（悬浮窗本身保留，仅文本不可见）

## 场景六

### 场景概述

恢复播放时，悬浮窗歌词淡入显示

### 场景逻辑步骤

- 1. 悬浮窗状态栏歌词开关已打开，当前处于暂停状态，悬浮窗歌词文本不可见
- 2. 用户恢复播放
- 3. 悬浮窗中的歌词文本以淡入动画显示，继续展示当前歌词行

## 场景七

### 场景概述

用户调整悬浮窗左右位置

### 场景逻辑步骤

- 1. 悬浮窗状态栏歌词开关已打开
- 2. 用户在设置页面拖动"左右位置"滑块，范围为0%到100%
- 3. 悬浮窗在屏幕水平方向上实时移动：0%表示最左侧，100%表示最右侧
- 4. 设置值持久化保存，下次启动时沿用该位置

## 场景八

### 场景概述

用户调整悬浮窗上下位置

### 场景逻辑步骤

- 1. 悬浮窗状态栏歌词开关已打开
- 2. 用户在设置页面拖动"上下位置"滑块，范围为0px到100px
- 3. 悬浮窗在屏幕垂直方向上实时移动：0px表示状态栏顶部，数值越大越向下偏移
- 4. 设置值持久化保存，下次启动时沿用该位置

## 场景九

### 场景概述

用户调整悬浮窗宽度

### 场景逻辑步骤

- 1. 悬浮窗状态栏歌词开关已打开
- 2. 用户在设置页面拖动"宽度"滑块，范围为50dp到750dp
- 3. 悬浮窗的歌词文本显示区域宽度实时变化
- 4. 宽度变化后，如果当前歌词文本超出新宽度则触发跑马灯滚动，如果未超出则静态展示
- 5. 设置值持久化保存，下次启动时沿用该宽度

## 场景十

### 场景概述

用户调整悬浮窗歌词字体大小

### 场景逻辑步骤

- 1. 悬浮窗状态栏歌词开关已打开
- 2. 用户在设置页面拖动"大小"滑块，范围为8dp到32dp
- 3. 悬浮窗中的歌词文本字体大小实时变化
- 4. 设置值持久化保存，下次启动时沿用该字体大小

## 场景十一

### 场景概述

用户选择悬浮窗歌词颜色

### 场景逻辑步骤

- 1. 悬浮窗状态栏歌词开关已打开
- 2. 用户在设置页面点击"选择颜色"，弹出颜色选择弹窗
- 3. 颜色选择弹窗中展示预设颜色列表供用户选择
- 4. 用户选择一种颜色后，悬浮窗中的歌词文本颜色实时更新为所选颜色
- 5. 设置值持久化保存，下次启动时沿用该颜色

## 场景十二

### 场景概述

用户打开歌词文本居中对齐开关

### 场景逻辑步骤

- 1. 悬浮窗状态栏歌词开关已打开
- 2. 用户在设置页面打开"歌词文本居中对齐"开关（默认为关闭）
- 3. 悬浮窗中的歌词文本对齐方式从左对齐切换为居中对齐
- 4. 歌词在设定宽度范围内居中展示
- 5. 设置值持久化保存，下次启动时沿用居中对齐

## 场景十三

### 场景概述

用户关闭歌词文本居中对齐开关，恢复左对齐

### 场景逻辑步骤

- 1. 悬浮窗状态栏歌词开关已打开
- 2. 用户在设置页面关闭"歌词文本居中对齐"开关
- 3. 悬浮窗中的歌词文本对齐方式从居中对齐切换为左对齐
- 4. 歌词在设定宽度范围内左对齐展示
- 5. 设置值持久化保存，下次启动时沿用左对齐

## 场景十四

### 场景概述

用户打开状态栏歌词不显示翻译开关，歌词仅展示原文

### 场景逻辑步骤

- 1. 悬浮窗状态栏歌词开关已打开
- 2. 用户在设置页面打开"状态栏歌词不显示翻译"开关（默认为关闭）
- 3. 当前播放歌曲存在翻译歌词时，悬浮窗仅展示歌词原文第一行，不展示翻译内容
- 4. 设置值持久化保存，下次启动时沿用该设置

## 场景十五

### 场景概述

用户关闭状态栏歌词不显示翻译开关，歌词展示原文和翻译

### 场景逻辑步骤

- 1. 悬浮窗状态栏歌词开关已打开
- 2. 用户在设置页面关闭"状态栏歌词不显示翻译"开关
- 3. 当前播放歌曲存在翻译歌词时，悬浮窗同时展示歌词原文和翻译内容
- 4. 设置值持久化保存，下次启动时沿用该设置

## 场景十六

### 场景概述

用户打开在播放界面隐藏开关，切换到播放页时悬浮窗歌词隐藏

### 场景逻辑步骤

- 1. 悬浮窗状态栏歌词开关已打开
- 2. 用户在设置页面打开"在播放界面隐藏"开关（默认为关闭）
- 3. 用户切换到播放页面
- 4. 悬浮窗歌词区域停止显示歌词（隐藏），避免与播放页面自身的歌词展示重复
- 5. 设置值持久化保存，下次启动时沿用该设置

## 场景十七

### 场景概述

用户打开在播放界面隐藏开关，离开播放页时悬浮窗歌词恢复显示

### 场景逻辑步骤

- 1. 悬浮窗状态栏歌词开关已打开，"在播放界面隐藏"开关已打开
- 2. 用户当前处于播放页面，悬浮窗歌词处于隐藏状态
- 3. 用户离开播放页面（返回其他页面）
- 4. 悬浮窗歌词恢复显示，继续展示当前播放歌词

## 场景十八

### 场景概述

应用服务启动时，如果悬浮窗状态栏歌词开关为打开状态，自动显示悬浮窗

### 场景逻辑步骤

- 1. 应用音乐播放服务启动
- 2. 读取持久化存储中的悬浮窗状态栏歌词开关状态
- 3. 如果开关为打开状态，自动创建悬浮窗并按照已保存的各项设置参数（位置、宽度、字体大小、颜色、对齐方式等）进行展示
- 4. 如果当前正在播放歌曲，立即开始展示歌词；否则等待播放开始

## 场景十九

### 场景概述

播放无歌词的歌曲时，悬浮窗不展示任何文本

### 场景逻辑步骤

- 1. 悬浮窗状态栏歌词开关已打开，悬浮窗已显示
- 2. 用户播放一首没有歌词的歌曲
- 3. 悬浮窗中不展示任何歌词文本，保持空白状态
- 4. 当切换到有歌词的歌曲时，悬浮窗恢复展示歌词
