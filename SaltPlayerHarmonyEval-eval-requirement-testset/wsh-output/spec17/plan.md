# 歌词翻译SPEC

在播放页歌词区域增加翻译按钮，支持中文到其他语言的翻译显示，切换翻译时包含歌词动效，mini歌词同步变化。

## 场景一

### 场景概述

用户在播放页歌词区域点击翻译按钮开启翻译，当前歌曲存在翻译歌词

### 场景逻辑步骤

- 1. 用户在播放页左滑进入歌词界面
- 2. 歌词界面底部设置栏右侧显示翻译按钮
- 3. 当前歌曲的歌词数据中包含翻译文本（即歌词行中存在副文本/翻译文本）
- 4. 翻译按钮以未激活样式显示（带边框的图标样式）
- 5. 用户点击翻译按钮
- 6. 翻译状态切换为开启，翻译按钮切换为激活样式（填充背景色，图标以镂空方式显示）
- 7. 歌词区域每行歌词下方以动效方式展示对应的翻译文本：
  - 翻译文本以透明度动画（300ms渐入）从不可见过渡到可见
  - 歌词行整体高度随翻译文本的出现而平滑变化
  - 当前正在播放的歌词行的翻译文本以完整透明度显示
  - 非当前播放的歌词行的翻译文本以降低透明度显示
- 8. 翻译开启状态持久化保存，下次打开应用时保持该状态

## 场景二

### 场景概述

用户在播放页歌词区域点击翻译按钮关闭翻译

### 场景逻辑步骤

- 1. 用户在播放页歌词界面，当前翻译状态为开启
- 2. 翻译按钮以激活样式显示（填充背景色）
- 3. 用户点击翻译按钮
- 4. 翻译状态切换为关闭，翻译按钮切换为未激活样式（带边框的图标样式）
- 5. 歌词区域的翻译文本以动效方式隐藏：
  - 翻译文本以透明度动画（300ms渐出）从可见过渡到不可见
  - 歌词行整体高度随翻译文本的消失而平滑收缩
- 6. 翻译关闭状态持久化保存

## 场景三

### 场景概述

用户在播放页歌词区域查看翻译按钮，但当前歌曲不存在翻译歌词

### 场景逻辑步骤

- 1. 用户在播放页左滑进入歌词界面
- 2. 当前歌曲的歌词数据中不包含翻译文本（即所有歌词行均无副文本）
- 3. 歌词界面底部设置栏右侧不显示翻译按钮
- 4. 歌词区域仅显示原文歌词，无翻译相关交互

## 场景四

### 场景概述

翻译开启状态下切换歌曲，新歌曲存在翻译歌词

### 场景逻辑步骤

- 1. 用户当前翻译状态为开启
- 2. 用户通过上一首、下一首、播放队列切歌或自动切歌等方式切换当前播放歌曲
- 3. 新歌曲的歌词数据中包含翻译文本
- 4. 歌词界面重新加载新歌曲歌词，翻译按钮继续显示为激活样式
- 5. 新歌曲的歌词区域同时显示原文歌词和翻译文本

## 场景五

### 场景概述

翻译开启状态下切换歌曲，新歌曲不存在翻译歌词

### 场景逻辑步骤

- 1. 用户当前翻译状态为开启
- 2. 用户通过上一首、下一首、播放队列切歌或自动切歌等方式切换当前播放歌曲
- 3. 新歌曲的歌词数据中不包含翻译文本
- 4. 歌词界面重新加载新歌曲歌词，翻译按钮隐藏
- 5. 歌词区域仅显示原文歌词
- 6. 翻译开启状态仍然保持，当切换到有翻译的歌曲时翻译按钮重新显示为激活样式

## 场景六

### 场景概述

翻译开启状态下，mini歌词区域同步显示翻译内容

### 场景逻辑步骤

- 1. 用户当前翻译状态为开启
- 2. 用户在播放页封面界面查看mini歌词区域（封面下方的迷你歌词显示区域）
- 3. mini歌词区域同步显示当前播放歌词行的翻译文本
- 4. mini歌词随播放进度实时滚动更新，翻译文本跟随原文歌词同步滚动

## 场景七

### 场景概述

翻译关闭状态下，mini歌词区域不显示翻译内容

### 场景逻辑步骤

- 1. 用户当前翻译状态为关闭
- 2. 用户在播放页封面界面查看mini歌词区域
- 3. mini歌词区域仅显示原文歌词，不显示翻译文本
- 4. mini歌词随播放进度实时滚动更新

## 场景八

### 场景概述

用户在歌词界面切换翻译状态后，mini歌词区域实时同步变化

### 场景逻辑步骤

- 1. 用户在播放页歌词界面点击翻译按钮切换翻译状态（开启或关闭）
- 2. 用户滑动回封面界面查看mini歌词区域
- 3. mini歌词区域的翻译显示状态与歌词界面保持一致：
  - 若翻译刚被开启，mini歌词区域显示翻译文本
  - 若翻译刚被关闭，mini歌词区域不显示翻译文本

## 场景九

### 场景概述

沉浸模式下翻译按钮隐藏，但翻译状态保持

### 场景逻辑步骤

- 1. 用户在播放页歌词界面进入沉浸模式
- 2. 歌词界面底部设置栏（包含翻译按钮）隐藏
- 3. 歌词区域的翻译显示状态保持不变（如果之前翻译是开启的，歌词区域仍然显示翻译文本）
- 4. 用户退出沉浸模式后，底部设置栏重新显示，翻译按钮恢复之前的状态

## 场景十

### 场景概述

歌词翻译切换时的歌词滚动动效

### 场景逻辑步骤

- 1. 用户在播放页歌词界面点击翻译按钮切换翻译状态
- 2. 由于翻译文本的显示或隐藏导致歌词行高度变化
- 3. 歌词列表平滑滚动到当前播放行，确保当前播放行在切换后仍然处于可视区域的正确位置
- 4. 滚动过程以动画方式完成，避免突兀跳转
