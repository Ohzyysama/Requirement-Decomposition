# 显示桌面歌词按钮SPEC

设置-通知中提供"显示桌面歌词按钮"开关，默认开启。开启后，通知栏、状态栏、实况窗对应的播放器卡片在上一曲按钮左侧增加歌词按钮，点击该按钮后将艺术家信息展示区域的内容替换为当前播放歌曲的歌词。

## 场景一

### 场景概述

开关默认开启，播放器卡片中上一曲按钮左侧显示歌词按钮

### 场景逻辑步骤

- 1. 用户首次安装应用或未修改过该设置，"显示桌面歌词按钮"开关默认为开启状态
- 2. 当有歌曲正在播放时，通知栏、状态栏、实况窗对应的播放器卡片中，在上一曲按钮的左侧显示歌词按钮
- 3. 播放器卡片的按钮排列顺序为：歌词按钮、上一曲按钮、播放/暂停按钮、下一曲按钮
- 4. 此时艺术家信息展示区域正常显示艺术家信息

## 场景二

### 场景概述

用户点击歌词按钮，艺术家信息展示区域替换为当前歌词

### 场景逻辑步骤

- 1. 当前播放歌曲存在歌词，且艺术家信息展示区域正在显示艺术家信息
- 2. 用户点击通知栏、状态栏或实况窗播放器卡片中的歌词按钮
- 3. 播放器卡片的艺术家信息展示区域的内容替换为当前播放进度对应的歌词文本
- 4. 歌词按钮的图标状态变更为"歌词已开启"状态
- 5. 随着歌曲播放进度推进，艺术家信息展示区域的歌词内容实时更新为当前播放进度对应的歌词

## 场景三

### 场景概述

用户再次点击歌词按钮，恢复显示艺术家信息

### 场景逻辑步骤

- 1. 当前艺术家信息展示区域正在显示歌词
- 2. 用户点击通知栏、状态栏或实况窗播放器卡片中的歌词按钮
- 3. 播放器卡片的艺术家信息展示区域的内容恢复为艺术家信息
- 4. 歌词按钮的图标状态变更为"歌词已关闭"状态

## 场景四

### 场景概述

歌词显示状态下切换歌曲，歌词内容同步更新

### 场景逻辑步骤

- 1. 当前艺术家信息展示区域正在显示歌词
- 2. 用户通过上一曲、下一曲或播放队列等方式切换当前播放歌曲
- 3. 如果新歌曲存在歌词，艺术家信息展示区域更新为新歌曲当前播放进度对应的歌词文本
- 4. 如果新歌曲不存在歌词，艺术家信息展示区域显示为空白或恢复显示新歌曲的艺术家信息

## 场景五

### 场景概述

当前歌曲无歌词时点击歌词按钮

### 场景逻辑步骤

- 1. 当前播放歌曲不存在歌词，艺术家信息展示区域正在显示艺术家信息
- 2. 用户点击歌词按钮
- 3. 由于当前歌曲无歌词，艺术家信息展示区域保持显示艺术家信息，不发生变化

## 场景六

### 场景概述

用户在设置中关闭"显示桌面歌词按钮"开关

### 场景逻辑步骤

- 1. 用户进入设置-通知页面
- 2. 用户将"显示桌面歌词按钮"开关从开启切换为关闭
- 3. 通知栏、状态栏、实况窗对应的播放器卡片中的歌词按钮立即消失
- 4. 如果关闭开关前艺术家信息展示区域正在显示歌词，则恢复显示艺术家信息
- 5. 播放器卡片的按钮排列顺序变为：上一曲按钮、播放/暂停按钮、下一曲按钮
- 6. 如果此时桌面歌词悬浮窗正在显示，弹出对话框询问用户是否同时关闭桌面歌词悬浮窗
  - 6.1 用户点击确认，关闭桌面歌词悬浮窗
  - 6.2 用户点击取消，桌面歌词悬浮窗保持显示

## 场景七

### 场景概述

用户在设置中开启"显示桌面歌词按钮"开关且已有悬浮窗权限

### 场景逻辑步骤

- 1. 用户进入设置-通知页面
- 2. 用户将"显示桌面歌词按钮"开关从关闭切换为开启
- 3. 系统检测到应用已具有悬浮窗权限
- 4. 开关成功切换为开启状态
- 5. 通知栏、状态栏、实况窗对应的播放器卡片中立即在上一曲按钮左侧显示歌词按钮

## 场景八

### 场景概述

用户在设置中开启"显示桌面歌词按钮"开关但无悬浮窗权限

### 场景逻辑步骤

- 1. 用户进入设置-通知页面
- 2. 用户将"显示桌面歌词按钮"开关从关闭切换为开启
- 3. 系统检测到应用没有悬浮窗权限
- 4. 弹出提示"无悬浮窗权限"，并跳转到系统悬浮窗权限设置页面
- 5. 如果用户在系统设置中授予了悬浮窗权限，返回应用后开关自动切换为开启状态，播放器卡片中显示歌词按钮
- 6. 如果用户在系统设置中未授予悬浮窗权限，返回应用后开关保持关闭状态，播放器卡片中不显示歌词按钮

## 场景九

### 场景概述

歌词按钮的开启状态在应用重启后保持

### 场景逻辑步骤

- 1. 用户开启"显示桌面歌词按钮"开关
- 2. 用户退出应用或应用被系统回收
- 3. 用户重新打开应用并播放歌曲
- 4. 播放器卡片中歌词按钮仍然显示，开关状态与退出前一致

## 场景十

### 场景概述

歌词按钮在播放器卡片紧凑视图中的显示

### 场景逻辑步骤

- 1. "显示桌面歌词按钮"开关为开启状态
- 2. 当播放器卡片以紧凑模式展示时（如通知栏折叠状态），紧凑视图中显示上一曲、播放/暂停、下一曲三个核心按钮
- 3. 歌词按钮不在紧凑视图的核心按钮中显示，仅在展开视图中可见
