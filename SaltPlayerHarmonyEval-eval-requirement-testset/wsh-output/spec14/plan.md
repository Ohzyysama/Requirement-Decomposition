# 音量平衡SPEC

设置-音频输出-音量平衡，通过读取音乐文件中嵌入的回放增益（ReplayGain）标签来控制内部播放音量，实现不同歌曲之间的音量一致性，非设备音量调节。

## 场景一

### 场景概述

用户开启音量平衡功能，播放含有回放增益标签的歌曲，内部音量根据标签值自动调整

### 场景逻辑步骤

- 1. 用户进入"设置-音频输出"页面，开启"音量平衡"开关（默认为开启状态）
- 2. 用户切换歌曲或重新播放当前歌曲
- 3. 系统读取当前播放歌曲文件中嵌入的回放增益标签（REPLAYGAIN_TRACK_GAIN 或 REPLAYGAIN_ALBUM_GAIN）
- 4. 系统将标签中的增益值（如"-3.5 dB"）转换为音量倍率
- 5. 系统将该音量倍率应用为内部播放音量，歌曲以调整后的音量播放
- 6. 播放界面显示当前回放增益值（如"RG -3.5 dB"）

## 场景二

### 场景概述

用户开启音量平衡功能，播放不含回放增益标签的歌曲，内部音量保持默认值

### 场景逻辑步骤

- 1. 用户进入"设置-音频输出"页面，确认"音量平衡"开关为开启状态
- 2. 用户切换歌曲或重新播放当前歌曲
- 3. 系统读取当前播放歌曲文件中的回放增益标签，未找到 REPLAYGAIN_TRACK_GAIN 和 REPLAYGAIN_ALBUM_GAIN 标签
- 4. 系统将内部播放音量保持为默认值（即不做任何音量调整，音量倍率为1.0）
- 5. 播放界面不显示回放增益值

## 场景三

### 场景概述

用户关闭音量平衡功能，无论歌曲是否含有回放增益标签，内部音量均保持默认值

### 场景逻辑步骤

- 1. 用户进入"设置-音频输出"页面，关闭"音量平衡"开关
- 2. 用户切换歌曲或重新播放当前歌曲
- 3. 系统不读取歌曲文件中的回放增益标签
- 4. 系统将内部播放音量保持为默认值（音量倍率为1.0），无论歌曲文件中是否嵌入了回放增益标签
- 5. 播放界面不显示回放增益值

## 场景四

### 场景概述

用户在播放过程中切换音量平衡开关状态，设置变更在切换歌曲或重新播放后生效

### 场景逻辑步骤

- 1. 用户正在播放歌曲
- 2. 用户进入"设置-音频输出"页面，切换"音量平衡"开关状态（从开启切换为关闭，或从关闭切换为开启）
- 3. 当前正在播放的歌曲音量不会立即发生变化
- 4. 用户切换到下一首歌曲或重新播放当前歌曲后，新的音量平衡设置生效

## 场景五

### 场景概述

歌曲文件同时包含专辑增益标签和音轨增益标签时，系统按标签读取顺序使用先找到的值

### 场景逻辑步骤

- 1. 用户开启"音量平衡"功能
- 2. 用户播放一首同时嵌入了 REPLAYGAIN_ALBUM_GAIN 和 REPLAYGAIN_TRACK_GAIN 标签的歌曲
- 3. 系统遍历歌曲文件的标签属性，使用先匹配到的增益值（REPLAYGAIN_ALBUM_GAIN 或 REPLAYGAIN_TRACK_GAIN，取决于文件中标签的存储顺序）
- 4. 系统根据该增益值调整内部播放音量
- 5. 播放界面显示所使用的回放增益值

## 场景六

### 场景概述

音量平衡功能调整的是内部播放音量，不影响设备系统音量

### 场景逻辑步骤

- 1. 用户开启"音量平衡"功能并播放含有回放增益标签的歌曲
- 2. 系统根据回放增益标签值调整内部播放音量倍率
- 3. 设备系统音量（媒体音量）保持不变，用户仍可通过设备音量键独立调节设备音量
- 4. 最终听到的音量为内部播放音量倍率与设备音量的叠加效果
