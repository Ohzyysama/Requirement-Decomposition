# 播放暂停淡入淡出SPEC

设置-音频输出中提供淡入淡出开关，开启后歌曲在播放和暂停状态切换时音量渐变过渡，切歌场景不涉及淡出效果。

## 场景一

### 场景概述

淡入淡出开关开启时，用户从暂停状态恢复播放，音量淡入

### 场景逻辑步骤

- 1. 用户在"设置-音频输出"页面开启"播放暂停时淡入淡出"开关（默认为开启状态）
- 2. 当前有歌曲处于暂停状态
- 3. 用户点击播放按钮恢复播放
- 4. 歌曲从当前暂停位置开始播放，音量从静音在500毫秒内平滑渐增至当前设定音量

## 场景二

### 场景概述

淡入淡出开关开启时，用户从播放状态暂停，音量淡出

### 场景逻辑步骤

- 1. 用户在"设置-音频输出"页面开启"播放暂停时淡入淡出"开关（默认为开启状态）
- 2. 当前有歌曲处于播放状态
- 3. 用户点击暂停按钮
- 4. 音量从当前设定音量在500毫秒内平滑渐减至静音
- 5. 音量渐减完成后，歌曲暂停在当前播放位置

## 场景三

### 场景概述

淡入淡出开关关闭时，用户从暂停状态恢复播放，音量立即恢复

### 场景逻辑步骤

- 1. 用户在"设置-音频输出"页面关闭"播放暂停时淡入淡出"开关
- 2. 当前有歌曲处于暂停状态
- 3. 用户点击播放按钮恢复播放
- 4. 歌曲从当前暂停位置开始播放，音量立即恢复至当前设定音量，无渐变过渡

## 场景四

### 场景概述

淡入淡出开关关闭时，用户从播放状态暂停，立即暂停

### 场景逻辑步骤

- 1. 用户在"设置-音频输出"页面关闭"播放暂停时淡入淡出"开关
- 2. 当前有歌曲处于播放状态
- 3. 用户点击暂停按钮
- 4. 歌曲立即暂停在当前播放位置，无音量渐变过渡

## 场景五

### 场景概述

切歌场景不触发淡出效果，无论淡入淡出开关是否开启

### 场景逻辑步骤

- 1. 无论"播放暂停时淡入淡出"开关处于开启或关闭状态
- 2. 当前有歌曲处于播放状态
- 3. 用户通过上一首、下一首或播放队列切换歌曲
- 4. 当前播放的歌曲立即停止，不执行淡出效果
- 5. 新歌曲加载完成后开始播放，若淡入淡出开关开启则新歌曲以淡入方式开始播放，若关闭则立即以设定音量播放

## 场景六

### 场景概述

淡入淡出开关实时生效，无需重启播放器

### 场景逻辑步骤

- 1. 用户在"设置-音频输出"页面切换"播放暂停时淡入淡出"开关状态
- 2. 开关状态变更立即生效，下一次播放或暂停操作即按新的开关状态执行
- 3. 无需重启应用或重新加载歌曲

## 场景七

### 场景概述

淡出过程中用户快速再次点击播放，淡入淡出操作互斥顺序执行

### 场景逻辑步骤

- 1. "播放暂停时淡入淡出"开关处于开启状态
- 2. 当前有歌曲处于播放状态，用户点击暂停按钮，音量开始淡出
- 3. 在淡出尚未完成时，用户再次点击播放按钮
- 4. 系统等待当前淡出操作完成并暂停后，再执行淡入操作恢复播放
- 5. 播放和暂停操作按顺序依次执行，不会出现音量状态混乱
