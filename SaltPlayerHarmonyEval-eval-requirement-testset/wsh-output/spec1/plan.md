# 在列表中显示封面SPEC

设置-用户界面中的"在列表中显示歌曲封面"开关，控制歌曲列表中是否显示歌曲封面图片。

## 场景一

### 场景概述

开关默认为打开状态，歌曲列表中显示歌曲封面

### 场景逻辑步骤

- 1. 用户首次使用应用或未修改过该设置
- 2. "在列表中显示歌曲封面"开关默认为打开状态
- 3. 用户进入任意包含歌曲列表的页面（如歌曲列表页、歌单详情页、专辑详情页、艺术家详情页、文件夹详情页等）
- 4. 每首歌曲的列表项中显示对应的歌曲封面图片

## 场景二

### 场景概述

用户关闭开关后，歌曲列表中不再显示歌曲封面

### 场景逻辑步骤

- 1. 用户进入"设置"页面，点击"用户界面"进入用户界面设置页面
- 2. 用户在"歌曲列表"分区下找到"在列表中显示歌曲封面"开关
- 3. 用户将开关从打开状态切换为关闭状态
- 4. 设置立即生效并持久化保存
- 5. 所有包含歌曲列表的页面（如歌曲列表页、歌单详情页、专辑详情页、艺术家详情页、文件夹详情页等）中的歌曲封面图片立即隐藏，列表项中不再显示封面

## 场景三

### 场景概述

用户重新打开开关后，歌曲列表中恢复显示歌曲封面

### 场景逻辑步骤

- 1. 用户进入"设置"页面，点击"用户界面"进入用户界面设置页面
- 2. 用户在"歌曲列表"分区下找到"在列表中显示歌曲封面"开关
- 3. 用户将开关从关闭状态切换为打开状态
- 4. 设置立即生效并持久化保存
- 5. 所有包含歌曲列表的页面中的歌曲封面图片立即恢复显示

## 场景四

### 场景概述

设置状态在应用重启后保持不变

### 场景逻辑步骤

- 1. 用户将"在列表中显示歌曲封面"开关切换为关闭状态
- 2. 用户退出应用并重新启动应用
- 3. 进入"设置-用户界面"页面，开关仍为关闭状态
- 4. 进入歌曲列表页面，歌曲封面仍然不显示
