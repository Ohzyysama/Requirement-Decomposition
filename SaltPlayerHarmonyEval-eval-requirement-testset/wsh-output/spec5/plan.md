# 圆形播放封面SPEC

设置-用户界面中提供圆形播放封面开关，默认关闭，打开后在播放页以圆形展示封面并自动旋转。

## 场景一

### 场景概述

用户在设置页面开启圆形播放封面开关，播放页封面切换为圆形并开始旋转

### 场景逻辑步骤

- 1. 用户进入"设置-用户界面"页面，在"播放界面"区域找到"圆形播放封面"开关
- 2. 该开关默认为关闭状态
- 3. 用户打开"圆形播放封面"开关，设置立即持久化保存
- 4. 播放页的封面立即从默认的圆角矩形切换为圆形展示
- 5. 如果当前正在播放歌曲，圆形封面立即开始匀速旋转，每25秒旋转一圈
- 6. 如果当前未在播放歌曲，圆形封面静止展示，不旋转

## 场景二

### 场景概述

用户在设置页面关闭圆形播放封面开关，播放页封面恢复为默认样式

### 场景逻辑步骤

- 1. 用户进入"设置-用户界面"页面，在"播放界面"区域找到"圆形播放封面"开关
- 2. 用户关闭"圆形播放封面"开关，设置立即持久化保存
- 3. 播放页的封面立即从圆形切换回默认的圆角矩形样式
- 4. 封面旋转动画停止，封面以静止状态展示

## 场景三

### 场景概述

圆形播放封面开启时，播放歌曲，封面开始旋转；暂停播放，封面停止旋转

### 场景逻辑步骤

- 1. 用户已开启"圆形播放封面"开关
- 2. 用户在播放页播放歌曲，圆形封面以匀速旋转，每25秒旋转一圈
- 3. 用户暂停播放，封面在当前旋转角度停止，保持当前朝向不变
- 4. 用户恢复播放，封面从暂停时的角度继续旋转，而非从初始角度重新开始

## 场景四

### 场景概述

圆形播放封面开启且正在播放时，切换到下一首/上一首歌曲，封面旋转重置

### 场景逻辑步骤

- 1. 用户已开启"圆形播放封面"开关，当前正在播放歌曲，封面正在旋转
- 2. 用户通过上一首、下一首、播放队列切歌或自动播放完毕切歌等方式切换歌曲
- 3. 封面旋转角度在250毫秒内平滑回归到初始角度（0度）
- 4. 回归初始角度后，封面重新开始匀速旋转

## 场景五

### 场景概述

圆形播放封面开启且暂停状态时，切换歌曲，封面旋转重置但不开始旋转

### 场景逻辑步骤

- 1. 用户已开启"圆形播放封面"开关，当前处于暂停状态，封面静止在某个角度
- 2. 用户通过上一首、下一首或播放队列等方式切换歌曲
- 3. 封面旋转角度在250毫秒内平滑回归到初始角度（0度）
- 4. 回归初始角度后，封面保持静止，不开始旋转

## 场景六

### 场景概述

圆形播放封面开启时，当前播放歌曲无封面图片，播放页不展示封面

### 场景逻辑步骤

- 1. 用户已开启"圆形播放封面"开关
- 2. 当前播放的歌曲没有封面图片数据
- 3. 播放页不展示任何封面内容（不展示占位图，不展示空白圆形）

## 场景七

### 场景概述

开启圆形播放封面后，"允许不规则封面"开关被禁用

### 场景逻辑步骤

- 1. 用户进入"设置-用户界面"页面
- 2. 用户打开"圆形播放封面"开关
- 3. "允许不规则封面"开关立即变为不可操作的禁用状态，无论其之前是开启还是关闭
- 4. 用户关闭"圆形播放封面"开关后，"允许不规则封面"开关恢复为可操作状态

## 场景八

### 场景概述

在迷你歌词模式下，圆形播放封面同样生效

### 场景逻辑步骤

- 1. 用户已开启"圆形播放封面"开关
- 2. 用户在播放页处于迷你歌词模式（封面缩小展示在歌词旁边）
- 3. 迷你歌词模式下的封面同样以圆形展示
- 4. 播放时圆形封面同样匀速旋转，暂停时停止旋转，切歌时重置角度，行为与普通模式一致

## 场景九

### 场景概述

在车载模式下，无论圆形播放封面开关状态如何，封面始终以圆形展示并旋转

### 场景逻辑步骤

- 1. 用户进入车载模式播放界面
- 2. 无论"圆形播放封面"开关是否开启，车载模式下的封面始终以圆形展示
- 3. 播放时封面自动旋转，暂停时停止旋转，行为与开启圆形播放封面时一致
