# 允许不规则封面SPEC

设置-用户界面-播放界面中提供"允许不规则封面"开关，控制播放页封面是以统一正方形裁剪展示还是以原始比例展示。

## 场景一

### 场景概述

开关默认开启，播放页封面以原始比例展示

### 场景逻辑步骤

- 1. 用户首次使用应用，"允许不规则封面"开关默认为开启状态
- 2. 用户进入播放页
- 3. 当前播放歌曲的封面图片以原始宽高比例展示，封面在播放页容器内按比例缩放适配：
  - 若封面宽度大于高度，则封面宽度填满容器宽度，高度按比例缩小
  - 若封面高度大于宽度，则封面高度填满容器高度，宽度按比例缩小
  - 若封面为正方形，则封面填满容器

## 场景二

### 场景概述

用户关闭开关，播放页封面以统一正方形裁剪展示

### 场景逻辑步骤

- 1. 用户进入设置-用户界面页面，在"播放界面"区域找到"允许不规则封面"开关
- 2. 用户将开关从开启切换为关闭
- 3. 设置立即生效并持久化保存
- 4. 用户进入播放页
- 5. 当前播放歌曲的封面图片以正方形裁剪方式展示，封面填满正方形容器，超出部分被裁剪

## 场景三

### 场景概述

用户开启开关，播放页封面恢复原始比例展示

### 场景逻辑步骤

- 1. 用户进入设置-用户界面页面，在"播放界面"区域找到"允许不规则封面"开关（当前为关闭状态）
- 2. 用户将开关从关闭切换为开启
- 3. 设置立即生效并持久化保存
- 4. 用户进入播放页
- 5. 当前播放歌曲的封面图片以原始宽高比例展示，不再进行正方形裁剪

## 场景四

### 场景概述

圆形播放封面开启时，不规则封面开关被禁用，封面强制以裁剪方式展示

### 场景逻辑步骤

- 1. 用户在设置-用户界面页面开启"圆形播放封面"开关
- 2. "允许不规则封面"开关变为禁用状态，用户无法操作该开关
- 3. 无论"允许不规则封面"开关之前处于何种状态，播放页封面均强制以裁剪方式填满圆形容器展示
- 4. 当用户关闭"圆形播放封面"开关后，"允许不规则封面"开关恢复为可操作状态，封面展示方式恢复为该开关对应的行为

## 场景五

### 场景概述

切换歌曲后，播放页封面按当前开关状态刷新展示

### 场景逻辑步骤

- 1. 用户在播放页通过上一首、下一首、播放队列切歌或自动切歌等方式切换当前播放歌曲
- 2. 播放页封面刷新为新歌曲的封面图片
- 3. 新封面按照当前"允许不规则封面"开关状态决定展示方式：
  - 开关开启：新封面以原始宽高比例展示
  - 开关关闭：新封面以正方形裁剪方式展示

## 场景六

### 场景概述

设置持久化，重启应用后保持用户选择的开关状态

### 场景逻辑步骤

- 1. 用户修改"允许不规则封面"开关状态（开启或关闭）
- 2. 用户退出应用并重新启动
- 3. 进入设置-用户界面页面，"允许不规则封面"开关保持用户上次设置的状态
- 4. 进入播放页，封面展示方式与用户上次设置一致
