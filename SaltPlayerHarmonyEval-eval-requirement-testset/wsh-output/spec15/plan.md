# 隐藏歌词界面控制面板SPEC

设置-用户界面-歌词界面中提供"隐藏歌词界面控制面板"开关，开启后播放页歌词页只展示歌曲标题、艺术家和歌词，隐藏底部控制区域（时间条、播放控制按钮、图标面板），扩大歌词展示区域。播放页歌词设置弹窗中同步提供该设置项。

## 场景一

### 场景概述

用户在设置-用户界面-歌词界面中开启"隐藏歌词界面控制面板"开关

### 场景逻辑步骤

- 1. 用户进入"设置"->"用户界面"->"歌词界面"页面
- 2. 页面中展示"隐藏歌词界面控制面板"开关项，该开关为Pro专属功能
- 3. 用户为Pro用户时，开关可操作；用户为非Pro用户时，开关不可操作
- 4. 用户点击开关，将其从关闭状态切换为开启状态
- 5. 设置立即生效并持久化存储，无需重启应用

## 场景二

### 场景概述

用户在设置-用户界面-歌词界面中关闭"隐藏歌词界面控制面板"开关

### 场景逻辑步骤

- 1. 用户进入"设置"->"用户界面"->"歌词界面"页面
- 2. 用户点击"隐藏歌词界面控制面板"开关，将其从开启状态切换为关闭状态
- 3. 设置立即生效并持久化存储，播放页歌词页恢复显示底部控制区域

## 场景三

### 场景概述

用户在播放页歌词设置弹窗中开启或关闭"隐藏歌词界面控制面板"开关

### 场景逻辑步骤

- 1. 用户在播放页歌词页底部点击歌词设置图标，弹出歌词设置弹窗
- 2. 歌词设置弹窗中包含与"设置-用户界面-歌词界面"页面相同的设置内容，包括"隐藏歌词界面控制面板"开关
- 3. 用户切换该开关的状态
- 4. 设置立即生效并持久化存储，与设置页面中的开关状态保持同步
- 5. 播放页歌词页的显示效果实时响应该设置变更

## 场景四

### 场景概述

"隐藏歌词界面控制面板"开启后，播放页歌词页的显示效果

### 场景逻辑步骤

- 1. 用户在播放页通过左右滑动切换到歌词页
- 2. 当"隐藏歌词界面控制面板"开关为开启状态时：
  - 2.1 播放页顶部区域正常展示歌曲标题和艺术家名称
  - 2.2 歌词展示区域向下扩大，底部间距缩小为固定小间距，歌词内容整体上移以利用释放的空间
  - 2.3 底部控制区域（包含时间进度条、播放/暂停/上一首/下一首按钮、图标面板）在用户从封面页滑动到歌词页的过程中，通过水平平移动画滑出屏幕
  - 2.4 当用户完全停留在歌词页时，底部控制区域完全不可见
- 3. 歌词页底部的歌词来源标签和翻译切换按钮的显示与否由沉浸模式控制，不受"隐藏歌词界面控制面板"设置影响

## 场景五

### 场景概述

"隐藏歌词界面控制面板"关闭后，播放页歌词页的显示效果

### 场景逻辑步骤

- 1. 用户在播放页通过左右滑动切换到歌词页
- 2. 当"隐藏歌词界面控制面板"开关为关闭状态时：
  - 2.1 播放页顶部区域正常展示歌曲标题和艺术家名称
  - 2.2 歌词展示区域底部保留控制面板高度的间距
  - 2.3 底部控制区域（时间进度条、播放控制按钮、图标面板）始终可见，不随页面滑动而隐藏
- 3. 歌词页底部的歌词来源标签和翻译切换按钮的显示与否由沉浸模式控制

## 场景六

### 场景概述

从封面页滑动到歌词页过程中，底部控制区域的动画过渡效果（开关开启时）

### 场景逻辑步骤

- 1. "隐藏歌词界面控制面板"开关为开启状态
- 2. 用户从封面页（中间页）向左滑动切换到歌词页（右侧页）
- 3. 在滑动过程中，底部控制区域根据滑动进度进行水平平移：
  - 3.1 从左向右布局时，控制区域向左方向平移，平移距离与滑动进度成正比
  - 3.2 从右向左布局时，控制区域向右方向平移，平移距离与滑动进度成正比
- 4. 当滑动完成到达歌词页时，控制区域完全移出屏幕可视范围
- 5. 当用户从歌词页滑回封面页时，控制区域随滑动进度逐渐回到原始位置

## 场景七

### 场景概述

"隐藏歌词界面控制面板"设置仅对竖屏普通播放界面生效

### 场景逻辑步骤

- 1. "隐藏歌词界面控制面板"开关为开启状态
- 2. 在竖屏普通播放界面（PlayerNormalUI）中，歌词页按照场景四的逻辑隐藏控制面板
- 3. 在横屏播放界面中，歌词页不受该设置影响，底部控制区域正常显示
- 4. 在全屏歌词界面中，歌词页不受该设置影响

## 场景八

### 场景概述

非Pro用户尝试操作"隐藏歌词界面控制面板"开关

### 场景逻辑步骤

- 1. 非Pro用户进入"设置"->"用户界面"->"歌词界面"页面，或在播放页打开歌词设置弹窗
- 2. "隐藏歌词界面控制面板"开关显示为不可操作状态（禁用态）
- 3. 开关旁展示Pro标识图标
- 4. 用户无法切换该开关的状态
