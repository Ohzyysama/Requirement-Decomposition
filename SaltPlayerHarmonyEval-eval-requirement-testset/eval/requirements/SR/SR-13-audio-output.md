# SR-13 音频输出
- SR-ID: SR-13
- 包含 AR: AR-13-01 音频焦点、AR-13-02 音量平衡、AR-13-03 播放暂停淡入淡出、AR-13-04 DSP 数字信号处理

| 字段 | 填写内容 |
|------|----------|
| [需求标题] | 音频输出 |
| [需求描述] | 本需求聚合应用音频输出与处理相关能力。通过 AR-13-01 进行音频焦点管理，在来电、其他应用抢占音频等场景下正确暂停、降低音量或恢复播放；通过 AR-13-02 提供左右声道音量平衡调节；通过 AR-13-03 在播放与暂停时进行淡入淡出处理，避免突兀的声音切换；通过 AR-13-04 提供数字信号处理（DSP）对输出音频进行处理。各项能力共同保障音频输出的稳定性、连贯性与听感质量。 |
| [需求价值] | 保障多应用并发与系统事件场景下的播放行为合理，并通过声道平衡、淡入淡出与 DSP 处理提升听感舒适度与音质表现。 |
| [需求场景] | 用户听歌时来电或其他应用抢占音频，应用按焦点策略暂停/降音并在恢复后续播；用户调节左右声道平衡；播放/暂停时音量平滑淡入淡出；通过 DSP 处理调整输出音频。 |
| [目标用户] | 对音质与听感有要求、以及在多应用并发场景下听歌的用户。 |
| [限制约束] | 音频焦点行为需遵循系统焦点策略，不可长期独占；淡入淡出与 DSP 处理不应引入明显延迟或破音；声道平衡调节范围需合理并可复位。 |
| [外部依赖] | 系统音频框架与音频焦点能力（AudioManager/AVSession 等）、系统音量与声道控制能力、本地音频设置持久化。 |
| [性能指标] | 评测基线目标：音频焦点事件响应 < 200ms，淡入淡出过渡平滑无破音，参数调节即时生效。 |
| [功耗指标] | 评测基线目标：息屏后台纯音频每小时电量 < 3%。 |
| [ROM&RAM] | 评测基线目标：安装包 < 100MB；运行内存 < 400MB。 |
| [验收标准] | 1. 来电或其他应用抢占音频时按焦点策略正确暂停/降音，结束后按策略恢复；2. 可调节左右声道音量平衡并即时生效、可复位；3. 播放与暂停时音量平滑淡入淡出、无突兀切换或破音；4. DSP 处理可对输出音频生效且不引入明显延迟。 |
| [验收设备] | HarmonyOS NEXT（API 12+）手机为主基线。 |
| [适用产品] | 手机 / 平板 / 车机（部分能力）。 |
| [适用产品差异分析] | 各端均支持音频焦点、声道平衡与淡入淡出；DSP 等处理能力在车机/平板受系统音频框架与硬件差异影响，可能降级或以系统默认处理为准。 |
| [视觉/生态/安全等扩展维度] | 音频焦点与音量行为需遵循系统音频生态规范，确保与系统及其他应用的并发行为合规、可预期，避免突发高音量等影响用户体验与安全。 |
