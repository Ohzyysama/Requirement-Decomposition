# AR-12-04 圆形播放封面
- AR-ID: AR-12-04
- 所属 SR: SR-12 主题与外观个性化

| 字段 | 填写内容 |
|------|----------|
| [需求标题] | 圆形播放封面 |
| [需求描述] | 提供开关让播放页的歌曲封面以圆形（唱片）样式展示，而非默认方形。开启后播放页封面裁剪为圆形显示，关闭则恢复方形。设置切换即时生效并在重启后保持。 |
| [需求价值] | 提供唱片式视觉风格，丰富播放页外观，满足个性化审美偏好。 |
| [需求场景] | 用户在外观设置中开启"圆形播放封面"，进入播放页查看封面以圆形展示。 |
| [目标用户] | 偏好唱片/圆形封面视觉风格的用户。 |
| [限制约束] | 仅影响播放页封面形状；圆形裁剪不应使封面内容明显失真或丢失关键画面。 |
| [外部依赖] | 封面图加载与裁剪能力、本地设置持久化。 |
| [性能指标] | 基线目标：形状切换即时生效，封面渲染无明显卡顿。 |
| [功耗指标] | 继承所属 SR。 |
| [ROM&RAM] | 继承所属 SR。 |
| [验收标准] | 1. 外观设置中存在"圆形播放封面"开关；2. 开启时播放页封面以圆形样式展示；3. 关闭时恢复为默认方形封面；4. 设置在应用重启后保持。 |
| [验收设备] | HarmonyOS NEXT（API 12+）手机为主基线。 |
| [适用产品] | 手机 / 平板 / 车机。 |
| [适用产品差异分析] | 各端播放页封面尺寸不同，圆形裁剪按对应尺寸适配。 |
| [视觉/生态/安全等扩展维度] | 圆形裁剪需居中保留主体画面，保证视觉美观。 |
