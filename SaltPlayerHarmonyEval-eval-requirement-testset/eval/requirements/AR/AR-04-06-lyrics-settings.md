# AR-04-06 歌词设置
- AR-ID: AR-04-06
- 所属 SR: SR-04 歌词显示

| 字段 | 填写内容 |
|------|----------|
| [需求标题] | 歌词设置 |
| [需求描述] | 提供一组歌词外观与交互的个性化设置项，包括：调节歌词字体大小、切换歌词居中对齐、调节字体粗细、开关歌词视图模糊效果（非当前行虚化），以及隐藏歌词界面的控制面板以获得更纯净的歌词视图。各项设置实时生效并持久化保存，下次进入歌词界面沿用上次配置。 |
| [需求价值] | 允许用户按个人偏好定制歌词外观与界面简洁度，提升歌词阅读舒适度与个性化体验。 |
| [需求场景] | 用户在歌词设置中调整字体大小/对齐/粗细/模糊或隐藏控制面板，歌词界面随设置即时更新。 |
| [目标用户] | 对歌词外观与界面有个性化需求的用户。 |
| [限制约束] | 字体大小/粗细在预设档位或合理范围内调节；各项设置相互独立；设置需本地持久化。 |
| [外部依赖] | 设置项偏好的本地持久化存储、歌词界面渲染能力。 |
| [性能指标] | 基线目标：调整任一设置后歌词界面即时刷新（<150ms），模糊效果渲染保持流畅。 |
| [功耗指标] | 继承所属 SR。 |
| [ROM&RAM] | 继承所属 SR。 |
| [验收标准] | 1. 调节字体大小后歌词字号即时变化并持久生效；2. 切换居中对齐后歌词在居中与默认对齐间切换；3. 调节字体粗细后歌词字重即时变化；4. 开启歌词视图模糊后非当前行虚化、当前行清晰；5. 隐藏控制面板后歌词界面仅保留歌词内容，再次进入沿用上次各项配置。 |
| [验收设备] | HarmonyOS NEXT（API 12+）手机为主基线。 |
| [适用产品] | 手机 / 平板 / 车机。 |
| [适用产品差异分析] | 设置项一致，字号档位与默认值可由各端屏幕尺寸微调。 |
| [视觉/生态/安全等扩展维度] | 模糊效果需兼顾视觉美观与性能；各设置在浅色/深色主题下均需保证可读性。 |
