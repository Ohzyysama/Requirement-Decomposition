# AR-07-03 隐藏指定文件夹
- AR-ID: AR-07-03
- 所属 SR: SR-07 文件夹与存储管理

| 字段 | 填写内容 |
|------|----------|
| [需求标题] | 隐藏指定文件夹 |
| [需求描述] | 用户可对文件夹列表中的某个文件夹执行「隐藏」操作，将其从文件夹列表中移除展示。被隐藏的文件夹及其下歌曲不再出现在文件夹列表与相关展示中。隐藏设置被持久化保存，重启应用后仍然生效。 |
| [需求价值] | 让用户屏蔽掉不希望在音乐列表中出现的目录（如铃声、录音、系统音效等），保持音乐列表的整洁。 |
| [需求场景] | 用户在文件夹列表中对某个不需要的目录选择「隐藏」，该文件夹随即从列表消失，重启应用后仍保持隐藏。 |
| [目标用户] | 希望从音乐展示中排除特定目录的用户。 |
| [限制约束] | 隐藏作用于指定文件夹整体；隐藏设置持久化保存；被隐藏文件夹及其歌曲不在文件夹列表/搜索结果中展示。 |
| [外部依赖] | 设置持久化存储、本地媒体扫描归类数据。 |
| [性能指标] | 评测基线目标：执行隐藏后列表即时刷新 < 100ms。 |
| [功耗指标] | 继承所属 SR。 |
| [ROM&RAM] | 继承所属 SR。 |
| [验收标准] | 1. 可对文件夹列表中的指定文件夹执行隐藏操作。 2. 隐藏后该文件夹及其下歌曲不再在文件夹列表与搜索结果中展示。 3. 隐藏设置被持久化保存，重启应用后仍生效。 |
| [验收设备] | HarmonyOS NEXT（API 12+）手机为主基线；平板/车机为扩展。 |
| [适用产品] | 手机 / 平板 / 车机。 |
| [适用产品差异分析] | 隐藏文件夹为通用能力，各形态一致；车机以简化交互为主。 |
| [视觉/生态/安全等扩展维度] | 视觉：文件夹项的隐藏操作入口；安全：隐藏仅影响展示范围，不删除任何本地文件。 |
