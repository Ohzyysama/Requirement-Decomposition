# AR-01-04 播放速度调节
- AR-ID: AR-01-04
- 所属 SR: SR-01 播放控制与队列

| 字段 | 填写内容 |
|------|----------|
| [需求标题] | 播放速度调节 |
| [需求描述] | 允许用户调节当前音频的播放速度，在保持音频可正常收听的前提下加快或放慢播放节奏。用户选择目标速度后，播放立即按新速度继续，速度设置在切歌后仍按产品策略保持一致的收听节奏。 |
| [需求价值] | 满足有声内容快听或细听等差异化收听需求，提升收听效率与灵活性。 |
| [需求场景] | 用户在播放界面打开速度调节控件，选择目标倍速，音频随即按新速度播放。 |
| [目标用户] | 有变速收听需求的用户。 |
| [限制约束] | 可选速度范围受系统多媒体播放能力支持区间约束；变速过程需保持音频连续不中断。 |
| [外部依赖] | 系统多媒体播放能力（变速播放）。 |
| [性能指标] | 速度切换即时生效，无明显卡顿或声音中断。 |
| [功耗指标] | 继承所属 SR。 |
| [ROM&RAM] | 继承所属 SR。 |
| [验收标准] | 1. 用户选择某一播放速度后，音频立即按该速度继续播放；2. 调速过程中播放保持连续，不出现可感知的中断或杂音；3. 当前生效的播放速度在界面上有明确显示；4. 速度设置在切换曲目后按产品策略正确延续。 |
| [验收设备] | HarmonyOS NEXT（API 12+）手机为主基线。 |
| [适用产品] | 手机 / 平板 / 车机。 |
| [适用产品差异分析] | 各端共用变速逻辑，速度选择控件形态随端适配。 |
| [视觉/生态/安全等扩展维度] | 变速控件需清晰呈现当前倍速档位，便于用户确认。 |
