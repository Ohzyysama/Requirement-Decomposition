# AR-03-07 歌曲更多操作菜单
- AR-ID: AR-03-07
- 所属 SR: SR-03 歌曲管理

| 字段 | 填写内容 |
|------|----------|
| [需求标题] | 歌曲更多操作菜单（添加到歌单/下一首播放/分享/跳转艺术家详情/跳转专辑详情/查看歌曲信息/从音乐库移除） |
| [需求描述] | 每个歌曲列表项的「更多」按钮打开单曲操作菜单，提供：添加到歌单、下一首播放、分享、跳转艺术家详情（该曲有多位艺术家时先弹出选择再跳转）、跳转专辑详情、查看歌曲信息、从音乐库移除。各菜单项针对当前这首歌生效，跳转类操作进入对应详情页，移除类操作对库内记录生效并给出反馈。 |
| [需求价值] | 在列表内对单首歌曲提供一站式操作入口，覆盖播放、归类、分享、跳转与信息查看，提升操作便捷性。 |
| [需求场景] | 用户点击某首歌的「更多」按钮，选择「下一首播放」插队，或选择「跳转艺术家详情」（多艺术家时先选一位）进入艺术家页。 |
| [目标用户] | 需要对单首本地歌曲执行播放、归类、分享或跳转的用户。 |
| [限制约束] | 菜单作用于当前单首歌曲；多艺术家时需先选择具体艺术家再跳转；从音乐库移除针对库内记录并需明确反馈。 |
| [外部依赖] | 歌单管理（SR-08，添加到歌单）、播放队列（SR-01，下一首播放）、系统分享能力、艺术家/专辑详情页（SR-06/SR-05）、歌曲信息详情（AR-03-08）、媒体库（移除）。 |
| [性能指标] | 评测基线目标：菜单展开与各操作响应流畅，跳转即时。 |
| [功耗指标] | 继承所属 SR。 |
| [ROM&RAM] | 继承所属 SR。 |
| [验收标准] | 1. 「更多」按钮打开菜单，含添加到歌单、下一首播放、分享、跳转艺术家详情、跳转专辑详情、查看歌曲信息、从音乐库移除等项。 2. 「下一首播放」将该曲插入播放队列下一首位置。 3. 跳转艺术家详情在多艺术家时先弹出选择再跳转，单艺术家时直接跳转；跳转专辑详情进入对应专辑页。 4. 「查看歌曲信息」打开该曲信息详情。 5. 「从音乐库移除」移除该曲库内记录并给出反馈。 |
| [验收设备] | HarmonyOS NEXT（API 12+）手机为主基线。 |
| [适用产品] | 手机 / 平板 / 车机。 |
| [适用产品差异分析] | 操作菜单为通用能力；车机受行车安全限制，建议精简菜单项并弱化分享/移除等编辑操作。 |
| [视觉/生态/安全等扩展维度] | 视觉：菜单项图标与文案清晰；生态：分享走系统分享通道；安全：从音乐库移除需明确反馈以避免误操作。 |
