# AR-01-03 播放队列查看/清空/移除/播放模式切换
- AR-ID: AR-01-03
- 所属 SR: SR-01 播放控制与队列

| 字段 | 填写内容 |
|------|----------|
| [需求标题] | 播放队列查看/清空/移除/播放模式切换 |
| [需求描述] | 提供当前播放队列的查看与管理能力。用户可打开播放队列查看待播放的全部曲目，一键清空整个队列，或从队列中单项移除某首歌曲。队列界面同时支持切换播放模式（循环/单曲/随机），便于用户在管理队列的同时调整播放顺序。点击队列中的曲目可直接切换播放该曲。 |
| [需求价值] | 让用户直观掌控当前播放队列内容与顺序，是连续收听场景下的关键管理入口。 |
| [需求场景] | 用户打开播放队列面板，查看曲目列表，执行清空、单项移除、点击播放或切换播放模式等操作。 |
| [目标用户] | 全体听歌用户。 |
| [限制约束] | 清空队列后应同步停止或重置当前播放上下文；移除当前正在播放曲目时需合理处理后续播放。 |
| [外部依赖] | 系统多媒体播放能力、本地偏好存储（队列与播放模式持久化）。 |
| [性能指标] | 队列面板打开与列表渲染流畅，长队列滚动不卡顿。 |
| [功耗指标] | 继承所属 SR。 |
| [ROM&RAM] | 继承所属 SR。 |
| [验收标准] | 1. 打开播放队列可看到当前待播放的全部曲目，并高亮标识正在播放项；2. 点击清空后队列被清空，播放上下文相应停止/重置；3. 对队列中任一曲目执行移除后，该曲目从队列消失且其余曲目顺序保持正确；4. 点击队列中某曲目可直接切换播放该曲；5. 在队列界面切换播放模式后，模式即时生效并影响后续选曲。 |
| [验收设备] | HarmonyOS NEXT（API 12+）手机为主基线。 |
| [适用产品] | 手机 / 平板 / 车机。 |
| [适用产品差异分析] | 平板可在更大画布展示队列；车机以简化操作为主，核心管理能力一致。 |
| [视觉/生态/安全等扩展维度] | 队列项的移除/清空操作建议给予明确的视觉反馈，避免误操作。 |
