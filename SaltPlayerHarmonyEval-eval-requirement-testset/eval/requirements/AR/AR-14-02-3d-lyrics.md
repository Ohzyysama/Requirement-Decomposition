# AR-14-02 立体歌词
- AR-ID: AR-14-02
- 所属 SR: SR-14 实验功能（实验室）

| 字段 | 填写内容 |
|------|----------|
| [需求标题] | 立体歌词 |
| [需求描述] | 在实验室中提供"立体歌词"（3D 歌词）预览能力，以带景深/立体感的方式呈现歌词，相比普通平面歌词更具视觉表现力。用户在实验室开启该功能后，播放界面的歌词以立体效果展示。该功能为预告/实验性质，效果与可用范围可能随版本调整。 |
| [需求价值] | 为追求沉浸式听歌体验的用户提供更具视觉冲击力的歌词呈现，丰富播放界面的表现形式。 |
| [需求场景] | 用户在实验室开启"立体歌词"开关后，进入播放页查看以立体效果渲染的歌词。 |
| [目标用户] | 追求沉浸式视觉体验、乐于尝鲜的用户。 |
| [限制约束] | 立体歌词为预告/实验性功能，效果、样式与可用性可能在后续版本调整或下线；仅在实验室开关开启时生效，默认关闭；对设备渲染能力有一定要求，低端设备上可能降级或不可用。 |
| [外部依赖] | 歌词解析与同步能力、播放页歌词渲染组件及图形/动画渲染能力。 |
| [性能指标] | 基线目标：开启后歌词滚动保持流畅（目标≥30fps），不引起明显掉帧或播放卡顿。 |
| [功耗指标] | 继承所属 SR。 |
| [ROM&RAM] | 继承所属 SR。 |
| [验收标准] | 1. 在实验室开启"立体歌词"后，播放页歌词以立体（3D）效果呈现；2. 关闭该开关后，歌词恢复为普通平面展示；3. 开关状态在应用重启后保持不变；4. 界面以"预告/实验性功能"形式明确标识该能力；5. 开启后歌词与音频保持同步，滚动流畅无明显卡顿。 |
| [验收设备] | HarmonyOS NEXT（API 12+）手机为主基线。 |
| [适用产品] | 手机 / 平板 / 车机。 |
| [适用产品差异分析] | 立体效果受各端屏幕尺寸与渲染能力影响，平板可呈现更大画面、车机需兼顾驾驶安全降低视觉干扰，具体效果由各端适配。 |
| [视觉/生态/安全等扩展维度] | 作为实验室预告功能呈现，需清晰标注"实验性/预览"；立体渲染应控制功耗与发热，并遵循系统图形渲染与无障碍可读性规范。 |
