# AR-07-07 文件夹详情-歌曲列表随机播放全部
- AR-ID: AR-07-07
- 所属 SR: SR-07 文件夹与存储管理

| 字段 | 填写内容 |
|------|----------|
| [需求标题] | 文件夹详情-歌曲列表随机播放全部 |
| [需求描述] | 在文件夹详情页面内，工具栏提供「随机播放」入口，入口旁显示该文件夹内的歌曲总数。用户点击后，系统将该文件夹下全部歌曲组成播放队列，从中随机选取一首作为起点开始播放，并把播放模式切换为随机播放。当该文件夹歌曲列表为空时，点击该入口不产生任何动作。 |
| [需求价值] | 让用户一键随机畅听某个文件夹内的全部歌曲，无需手动挑选。 |
| [需求场景] | 用户进入某文件夹详情页，点击「随机播放（共 N 首）」，应用随机选一首开始播放并进入随机播放模式。 |
| [目标用户] | 希望随机畅听某个文件夹内全部歌曲的用户。 |
| [限制约束] | 文件夹歌曲列表为空时不响应；随机播放会以该文件夹全部歌曲为队列并切换为随机播放模式。 |
| [外部依赖] | 音频播放服务（设置队列与播放模式）、本地已扫描歌曲数据。 |
| [性能指标] | 评测基线目标：点击到开始播放即时响应 < 100ms。 |
| [功耗指标] | 继承所属 SR。 |
| [ROM&RAM] | 继承所属 SR。 |
| [验收标准] | 1. 文件夹详情页内「随机播放」入口旁显示该文件夹歌曲总数。 2. 点击后以文件夹内全部歌曲为队列、随机选一首起播，并切换为随机播放模式。 3. 文件夹歌曲列表为空时点击不产生任何动作。 |
| [验收设备] | HarmonyOS NEXT（API 12+）手机为主基线；平板/车机为扩展。 |
| [适用产品] | 手机 / 平板 / 车机。 |
| [适用产品差异分析] | 随机播放为通用能力，各形态一致；车机可作为主要的快捷播放入口。 |
| [视觉/生态/安全等扩展维度] | 安全：仅作用于该文件夹内本地已扫描歌曲。 |
