# AR-03-06 歌曲列表多选操作（播放选中队列/添加到歌单/从音乐库移除）
- AR-ID: AR-03-06
- 所属 SR: SR-03 歌曲管理

| 字段 | 填写内容 |
|------|----------|
| [需求标题] | 歌曲列表多选操作（播放选中队列/添加到歌单/从音乐库移除） |
| [需求描述] | 歌曲列表支持进入多选模式，用户通过勾选框选中一首或多首歌曲，并提供全选/反选能力。选中后可对这批歌曲执行批量操作：播放选中歌曲（以选中项构建播放队列并播放）、添加到歌单、从音乐库移除。退出多选模式后列表恢复正常浏览交互。 |
| [需求价值] | 让用户一次性对多首歌曲批量播放、归类或清理，避免逐首重复操作，提升管理效率。 |
| [需求场景] | 用户长按或点击多选入口进入多选模式，勾选若干歌曲后点击「播放选中」「添加到歌单」或「从音乐库移除」完成批量处理。 |
| [目标用户] | 需要批量整理或批量播放本地歌曲的用户。 |
| [限制约束] | 批量操作作用于已勾选的歌曲；从音乐库移除为针对库内记录的移除操作，需有明确反馈；未选中任何歌曲时批量操作不可用。 |
| [外部依赖] | 播放控制与队列能力（SR-01，播放选中队列）、歌单管理能力（SR-08，添加到歌单）、媒体库（移除库内记录）。 |
| [性能指标] | 评测基线目标：进入/退出多选与勾选响应流畅；批量操作即时反馈。 |
| [功耗指标] | 继承所属 SR。 |
| [ROM&RAM] | 继承所属 SR。 |
| [验收标准] | 1. 可进入多选模式并通过勾选框选中多首歌曲，支持全选/反选。 2. 「播放选中」以选中歌曲构建队列并播放。 3. 「添加到歌单」可将选中歌曲批量加入目标歌单。 4. 「从音乐库移除」可批量移除选中歌曲并给出反馈。 5. 退出多选模式后列表恢复正常浏览交互。 |
| [验收设备] | HarmonyOS NEXT（API 12+）手机为主基线。 |
| [适用产品] | 手机 / 平板 / 车机。 |
| [适用产品差异分析] | 多选为通用能力；车机受行车安全限制，建议弱化或简化批量编辑类操作。 |
| [视觉/生态/安全等扩展维度] | 视觉：多选模式下顶部显示选中数量与批量操作入口；安全：从音乐库移除为可感知的批量删除，需明确反馈避免误删。 |
