# AR-07-06 文件夹详情-歌曲列表 ABC 快速索引（含定位到当前歌曲）
- AR-ID: AR-07-06
- 所属 SR: SR-07 文件夹与存储管理

| 字段 | 填写内容 |
|------|----------|
| [需求标题] | 文件夹详情-歌曲列表 ABC 快速索引（含定位到当前歌曲） |
| [需求描述] | 在文件夹详情页面内，当歌曲列表按标题、文件名、专辑或艺术家排序时，列表右侧显示一条 A 到 Z（并含数字与「#」）的字母索引条；其它排序方式下不显示。用户点击或沿索引条滑动到某字母时，列表跳转到该字母（按拼音首字母）下的第一首歌，并弹出当前字母大字提示伴随轻微震动反馈。页面还提供「定位到当前播放歌曲」能力，可将列表滚动到该文件夹内正在播放的歌曲位置。 |
| [需求价值] | 在文件夹歌曲较多时让用户通过首字母快速跳转定位，并能一键回到当前播放的歌曲，免去长距离滚动。 |
| [需求场景] | 用户在某文件夹详情页沿右侧字母条滑动到字母「W」，列表跳到以 W 开头的第一首歌；或点击定位按钮直接滚动到当前播放的歌曲。 |
| [目标用户] | 文件夹内歌曲较多、需要快速跳转定位的用户。 |
| [限制约束] | 索引条仅在按标题/文件名/专辑/艺术家排序且列表非空时显示；跳转按拼音首字母匹配；定位以该文件夹内当前播放歌曲为目标，无当前歌曲时不定位。 |
| [外部依赖] | 系统拼音转写能力、系统震动服务、当前播放状态。 |
| [性能指标] | 评测基线目标：点击/滑动字母或定位到列表跳转即时生效 < 100ms。 |
| [功耗指标] | 继承所属 SR。 |
| [ROM&RAM] | 继承所属 SR。 |
| [验收标准] | 1. 文件夹详情页内，仅在按标题/文件名/专辑/艺术家排序且列表非空时右侧显示 A-Z（含数字与 #）索引条。 2. 点击或滑动某字母时，列表跳转到该字母（按拼音首字母）下第一首歌，并弹出字母提示伴随震动。 3. 提供「定位到当前播放歌曲」能力，可将列表滚动到该文件夹内正在播放的歌曲。 |
| [验收设备] | HarmonyOS NEXT（API 12+）手机为主基线；平板/车机为扩展。 |
| [适用产品] | 手机 / 平板 / 车机。 |
| [适用产品差异分析] | 字母索引与定位为通用能力，各形态一致；车机因驾驶安全可弱化滑动索引。 |
| [视觉/生态/安全等扩展维度] | 视觉：高亮色大字字母提示气泡、定位按钮；生态：复用系统拼音与震动能力；安全：仅作用于本地已扫描数据。 |
