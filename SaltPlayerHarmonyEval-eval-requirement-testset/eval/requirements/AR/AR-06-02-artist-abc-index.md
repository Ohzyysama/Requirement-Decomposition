# AR-06-02 艺术家列表 ABC 快速索引
- AR-ID: AR-06-02
- 所属 SR: SR-06 艺术家管理

| 字段 | 填写内容 |
|------|----------|
| [需求标题] | 艺术家列表 ABC 快速索引 |
| [需求描述] | 艺术家列表右侧提供 ABC 字母快速索引条，用户点击或滑动某个字母即可将列表快速定位到以该字母开头的艺术家分组。索引依据艺术家名首字母（含中文拼音首字母）归类，便于在大量艺术家中快速跳转。 |
| [需求价值] | 在艺术家数量较多时，让用户通过字母索引快速跳转目标分组，避免长距离滚动，提升查找效率。 |
| [需求场景] | 用户面对大量艺术家，点击或滑动右侧 ABC 索引条上的某字母，列表立即滚动到对应字母分组的起始位置。 |
| [目标用户] | 需在大量艺术家中快速定位的普通用户。 |
| [限制约束] | 索引按艺术家名首字母（中文取拼音首字母）归类；无对应分组的字母可不响应或定位至最近分组。 |
| [外部依赖] | 艺术家名拼音/首字母解析、艺术家列表分组数据。 |
| [性能指标] | 评测基线目标：点击字母定位响应即时（无明显卡顿）；滚动 ≥ 60fps。 |
| [功耗指标] | 继承所属 SR。 |
| [ROM&RAM] | 继承所属 SR。 |
| [验收标准] | 1. 列表右侧展示 ABC 字母快速索引条。 2. 点击或滑动某字母时，列表定位到该字母开头的艺术家分组。 3. 中文艺术家名按拼音首字母正确归类。 |
| [验收设备] | HarmonyOS NEXT（API 12+）手机为主基线；平板/车机为扩展。 |
| [适用产品] | 手机 / 平板 / 车机。 |
| [适用产品差异分析] | 快速索引为通用能力，各形态一致；车机可适当放大索引条点击区域。 |
| [视觉/生态/安全等扩展维度] | 视觉：右侧悬浮字母索引条、定位时分组标识；交互：支持点击与滑动定位。 |
