# AR-03-04 歌曲列表 ABC 快速索引（含定位到当前歌曲）
- AR-ID: AR-03-04
- 所属 SR: SR-03 歌曲管理

| 字段 | 填写内容 |
|------|----------|
| [需求标题] | 歌曲列表 ABC 快速索引（含定位到当前歌曲） |
| [需求描述] | 歌曲列表在侧边提供 A-Z 字母快速索引条，用户点击或滑动某个字母即可将列表快速定位到对应首字母的歌曲分组。索引条还提供「定位到当前播放歌曲」能力，一键将列表滚动到正在播放的歌曲位置，便于在长列表中快速回到当前曲目。 |
| [需求价值] | 在大量歌曲中通过字母索引与定位当前歌曲快速跳转，显著减少滚动操作，提升导航效率。 |
| [需求场景] | 用户在很长的歌曲列表中点击侧边字母「L」直接跳到 L 开头的歌曲，或点击定位按钮快速回到正在播放的那首歌。 |
| [目标用户] | 本地歌曲数量较多、需要在长列表中快速跳转的用户。 |
| [限制约束] | 字母索引依据歌曲首字母分组，索引有效性与当前排序方式相关；定位当前歌曲需存在正在播放的曲目。 |
| [外部依赖] | 媒体库扫描数据（首字母分组依据）、当前播放状态（定位当前歌曲的目标）、歌曲列表排序（AR-03-03）。 |
| [性能指标] | 评测基线目标：点击字母或定位按钮后跳转即时响应，无可感知卡顿。 |
| [功耗指标] | 继承所属 SR。 |
| [ROM&RAM] | 继承所属 SR。 |
| [验收标准] | 1. 侧边提供 A-Z 字母索引条，点击/滑动字母定位到对应首字母分组。 2. 提供定位到当前播放歌曲的能力，一键滚动到该歌曲。 3. 无正在播放歌曲时定位操作有合理的无效或提示反馈。 4. 索引与定位在长列表下均能正确跳转。 |
| [验收设备] | HarmonyOS NEXT（API 12+）手机为主基线。 |
| [适用产品] | 手机 / 平板 / 车机。 |
| [适用产品差异分析] | 快速索引为通用能力；车机受触控精度限制，建议加大字母触控区域或简化为定位能力。 |
| [视觉/生态/安全等扩展维度] | 视觉：拖动索引时显示当前字母气泡提示；安全：仅在本地列表内跳转。 |
