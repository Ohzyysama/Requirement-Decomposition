# AR-07-09 文件夹详情-歌曲更多操作菜单（添加到歌单/下一首播放/分享/跳转艺术家详情(多艺术家先选择)/跳转专辑详情/查看歌曲信息/从音乐库移除）
- AR-ID: AR-07-09
- 所属 SR: SR-07 文件夹与存储管理

| 字段 | 填写内容 |
|------|----------|
| [需求标题] | 文件夹详情-歌曲更多操作菜单（添加到歌单/下一首播放/分享/跳转艺术家详情(多艺术家先选择)/跳转专辑详情/查看歌曲信息/从音乐库移除） |
| [需求描述] | 在文件夹详情页面内，用户点击某首歌右侧的「更多」按钮可打开该曲操作菜单，菜单提供：添加到歌单、下一首播放、分享、跳转艺术家详情（当该曲存在多个艺术家时先弹出选择再跳转）、跳转专辑详情、查看歌曲信息、从音乐库移除等操作。各操作对应跳转相应页面或执行相应动作。 |
| [需求价值] | 让用户在文件夹详情页内对单首歌快速发起常用操作，无需进入其它页面即可完成播放、整理与跳转。 |
| [需求场景] | 用户在某文件夹详情页点击一首歌的「更多」按钮，选择「下一首播放」「跳转专辑详情」或「查看歌曲信息」等完成操作。 |
| [目标用户] | 需要对文件夹内单首歌进行播放、整理与跳转操作的用户。 |
| [限制约束] | 跳转艺术家详情时若该曲有多个艺术家，需先选择具体艺术家；「从音乐库移除」仅移出展示、不删除本地文件；分享依赖系统分享能力。 |
| [外部依赖] | 歌单数据、音频播放服务、系统分享能力、艺术家/专辑详情页、歌曲信息面板、媒体库数据。 |
| [性能指标] | 评测基线目标：菜单打开与操作触发即时响应 < 100ms。 |
| [功耗指标] | 继承所属 SR。 |
| [ROM&RAM] | 继承所属 SR。 |
| [验收标准] | 1. 文件夹详情页内点击「更多」可打开单曲操作菜单。 2. 菜单提供添加到歌单、下一首播放、分享、跳转艺术家详情、跳转专辑详情、查看歌曲信息、从音乐库移除。 3. 跳转艺术家详情时，多艺术家情况下先弹出选择再跳转。 4. 「从音乐库移除」仅移出展示、不删除本地文件。 |
| [验收设备] | HarmonyOS NEXT（API 12+）手机为主基线；平板/车机为扩展。 |
| [适用产品] | 手机 / 平板 / 车机。 |
| [适用产品差异分析] | 单曲菜单为通用能力；车机因驾驶安全通常精简菜单项与交互。 |
| [视觉/生态/安全等扩展维度] | 视觉：弹出式操作菜单，多艺术家时二次选择弹窗；生态：复用系统分享能力；安全：移除仅作用于音乐库展示，不删除本地文件。 |
