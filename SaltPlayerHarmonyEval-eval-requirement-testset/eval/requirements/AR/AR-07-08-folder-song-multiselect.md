# AR-07-08 文件夹详情-歌曲列表多选操作（播放选中队列/添加到歌单/从音乐库移除）
- AR-ID: AR-07-08
- 所属 SR: SR-07 文件夹与存储管理

| 字段 | 填写内容 |
|------|----------|
| [需求标题] | 文件夹详情-歌曲列表多选操作（播放选中队列/添加到歌单/从音乐库移除） |
| [需求描述] | 在文件夹详情页面内，用户可进入多选模式（例如长按某首歌进入），列表每项右侧出现勾选框并可勾选多首歌曲，顶部显示已选数量与全选/取消全选。针对选中的歌曲可执行批量操作：将选中歌曲作为播放队列播放、把选中歌曲添加到指定歌单、把选中歌曲从音乐库移除。操作完成后退出多选并清空勾选状态。 |
| [需求价值] | 让用户在文件夹详情页内一次性处理多首歌曲，提升批量播放、整理与歌单管理的效率。 |
| [需求场景] | 用户在某文件夹详情页长按进入多选，勾选若干首歌，选择「播放选中」「添加到歌单」或「从音乐库移除」完成批量操作。 |
| [目标用户] | 需要在文件夹内批量管理歌曲的用户。 |
| [限制约束] | 批量操作作用于当前勾选的歌曲集合；「从音乐库移除」仅移出音乐库展示，不删除本地文件；操作后退出多选并清空选择。 |
| [外部依赖] | 音频播放服务（播放选中队列）、歌单数据（添加到歌单）、媒体库数据（移除）。 |
| [性能指标] | 评测基线目标：批量操作即时生效 < 100ms（不含大批量歌单写入耗时）。 |
| [功耗指标] | 继承所属 SR。 |
| [ROM&RAM] | 继承所属 SR。 |
| [验收标准] | 1. 文件夹详情页内可进入多选并勾选多首歌曲。 2. 可将选中歌曲作为播放队列播放。 3. 可将选中歌曲添加到指定歌单。 4. 可将选中歌曲从音乐库移除（仅移出展示、不删除本地文件）。 5. 操作完成后退出多选并清空勾选状态。 |
| [验收设备] | HarmonyOS NEXT（API 12+）手机为主基线；平板/车机为扩展。 |
| [适用产品] | 手机 / 平板 / 车机。 |
| [适用产品差异分析] | 多选批量操作为通用能力；车机因驾驶安全通常弱化或不提供多选交互。 |
| [视觉/生态/安全等扩展维度] | 视觉：顶部多选工具栏与列表勾选框；安全：移除仅作用于音乐库展示，不删除本地文件。 |
